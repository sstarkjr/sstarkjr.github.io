A living list of the tools, languages, and services I use regularly. Inspired by [uses.this](https://usesthis.com/).

## Languages & Query

- **Python** — primary language for everything: data processing, ML, LLM pipelines, automation
- **SQL** — daily driver for data exploration and transformation
- **PySpark** — distributed processing on large datasets
- **R** — occasional statistical analysis

## Machine Learning & AI

- **OpenAI API** — GPT-4o, text-embedding-3-large, batch processing, vector stores
- **Google Gemini / Vertex AI** — LLM pipelines and document intelligence on GCP
- **scikit-learn** — clustering, classification, regression
- **UMAP** — dimensionality reduction for embeddings visualization
- **HDBSCAN** — density-based clustering for NLP and topic modeling
- **Sentence Transformers** — semantic embeddings

## Data & Cloud

- **Google BigQuery** — primary data warehouse
- **Google Cloud Platform (GCP)** — cloud storage, pipelines, Vertex AI
- **Pandas** — standard data manipulation
- **Plotly** — interactive visualizations

## Development

- **Jupyter** — exploratory analysis and prototyping
- **VS Code** — primary editor
- **Git / GitHub** — version control
- **Docker** — containerized development environments

## Visualization

- **Tableau** — dashboards and business reporting
- **Plotly** — interactive charts and embedded visualizations
- **matplotlib** — static plots and quick EDA

## Photography

- **DJI Mavic 3** (Hasselblad L2D-20c) — drone, aerial photography
- **Nikon DSLR** — ground photography
- **Adobe Lightroom Classic** — editing and photo management
- **Adobe Photoshop** — compositing and advanced edits
